Kenapa tidak bisa langsung dari browser?
WhatsApp tidak mengizinkan koneksi langsung dari halaman web. Diperlukan server backend (Node.js/Python) yang menjalankan library WhatsApp untuk membaca QR code dan berkomunikasi dengan server WhatsApp. Dashboard ini adalah frontend yang mengontrol backend tersebut.

whatsapp-web.js / Baileys

Library Node.js open-source. Scan QR dari WhatsApp Web seperti biasa. Gratis dan tanpa batasan.

Rekomendasi

WhatsApp Cloud API

API resmi dari Meta. Butuh bisnis terverifikasi. Stabil tapi berbayar per pesan.

Langkah Implementasi (whatsapp-web.js)

Ikuti langkah berikut untuk membangun backend bot

1

Install Node.js dan inisialisasi project

Buat folder project, lalu jalankan:
npm init -y
npm install whatsapp-web.js express socket.io qrcode

2

Buat file server.js

Buat server Express + Socket.IO yang menginisialisasi klien WhatsApp. Saat pertama kali dijalankan, library akan menghasilkan QR code. Scan QR tersebut dari aplikasi WhatsApp di ponsel (Menu > Perangkat Tertaut > Tautkan Perangkat).

3

Implementasi logika auto-reply

Di event listener message, cocokkan isi pesan dengan kata kunci yang tersimpan di database. Kirim balasan menggunakan client.sendMessage(). Dashboard ini mengirim update aturan ke backend via WebSocket.

4

Hubungkan dashboard ke backend

Dashboard ini (HTML) terhubung ke backend via Socket.IO. Saat aturan ditambah/diubah di dashboard, dikirim ke backend. Saat pesan masuk di backend, dikirim ke dashboard untuk ditampilkan di log. Ubah URL WebSocket di kode untuk menunjuk ke server Anda.

5

Jalankan server

node server.js — Buka dashboard di browser, buka halaman Koneksi, scan QR code yang muncul. Setelah terhubung, toggle Bot ke ON di sidebar.

Dalam implementasi nyata, QR code dihasilkan oleh backend. Tombol ini mensimulasikan proses scanning untuk demo.
